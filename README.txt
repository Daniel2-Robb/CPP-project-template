Controls:
	Menu:
		Arrow Key Up / Arrow Key Down - move cursor
		Enter - Select option highlighted by cursor
		Left Mouse Button - Select option highlighted by mouse
		
	In-Game:
		Escape - Exit game
		Left Mouse Button / Enter - Move to next screen
		Left Mouse Button (with Judgement tab open) - select highlighted judgement
		Left Mouse Button (Held) - Drag Passport
		Right Mouse Button - Open Judgement tab
